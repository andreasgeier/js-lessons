JavaScript Cheat Sheet

String methods

at()

const str = 'Hello World'

str.at(1) // => 'e'
str.at(-2) // => 'l'
str.at(99) // => undefined

The at() method takes an integer value and returns a new String consisting of the single UTF-16 code unit located at the specified offset. This method allows for positive and negative integers. Negative integers count back from the last string character.

Syntax: at(index) / Return value: String

charAt()

const str = 'Hello World'

str.charAt(1) // => 'e'
str.charAt(99) // => ''

The charAt() method of String values returns a new string consisting of the single UTF-16 code unit located at the specified offset into the string.

Syntax: charAt(index) / Return value: String

concat()

const str1 = 'Hello'
const str2 = 'World'

str1.concat(' ', str2) // => 'Hello World'

The concat() method concatenates the string arguments to the calling string and returns a new string.

Syntax: concat(string) concat(string1, string2, /* ..., */ stringN) / Return value: String

endsWith()

const str = 'Hello World'

str.endsWith('World') // => true
str.endsWith('World', 5) // => false

The endsWith() method determines whether a string ends with the characters of a specified string, returning true or false as appropriate.

Syntax: endsWith(searchString) endsWith(searchString, endPosition) / Return value: Boolean

includes()

const str = 'Hello World'

str.includes('Hello') // => true
str.includes('Hello', 5) // => false

The includes() method performs a case-sensitive search to determine whether one string may be found within another string, returning true or false as appropriate.

Syntax: includes(searchString) includes(searchString, startPosition) / Return value: Boolean

indexOf()

const str = 'Hello World'

str.indexOf('lo') // => 3
str.indexOf('lo', 5) // => -1

The indexOf() method of String values searches this string and returns the index of the first occurrence of the specified substring. It takes an optional starting position and returns the first occurrence of the specified substring at an index greater than or equal to the specified number.

Syntax: indexOf(searchString) indexOf(searchString, startPosition) / Return value: Found index as Integer, or -1 if not found

lastIndexOf()

const str = 'My dog is not like other dogs.'

str.lastIndexOf('dog') // => 24
str.lastIndexOf('dog', 15) // => 3

The lastIndexOf() method of String values searches this string and returns the index of the last occurrence of the specified substring. It takes an optional starting position and returns the last occurrence of the specified substring at an index less than or equal to the specified number.

Syntax: lastIndexOf(searchString) lastIndexOf(searchString, endPosition) / Return value: Found index, or -1 if not found

match()

const str = 'Hello World'

str.match(/[A-Z]/g) // => ['H', 'W']
str.match(/[A-Z]/) // => ['H']
str.match(/[0-9]/) // => null

The match() method retrieves the result of matching a string against a regular expression (regex).

Syntax: match(regex) / Return value: Array

padEnd()

const str = 'abc'

str.padEnd(8) // => 'abc     '
str.padEnd(8, '.') // => 'abc.....'

The padEnd() method pads the current string with a given string (repeated, if needed) so that the resulting string reaches a given length. The padding is applied from the end of the current string.

Syntax: padEnd(targetLength) padEnd(targetLength, padString) / Return value: String

padStart()

const last4DigitsOfCreditCard = '6789'

last4DigitsOfCreditCard.padStart(16, '*') // => '************6789'

The padStart() method pads the current string with another string (multiple times, if needed) until the resulting string reaches the given length. The padding is applied from the start of the current string.

Syntax: padStart(targetLength) padStart(targetLength, padString) / Return value: String

repeat()

const str = 'Hello World! '

str.repeat(2) // => 'Hello World! Hello World! '

The repeat() method constructs and returns a new string which contains the specified number of copies of the string on which it was called, concatenated together.

Syntax: repeat(count) / Return value: String

replace()

const str = 'My dog is not like other dogs.'

str.replace('dog', 'cat') // => 'My cat is not like other dogs.'
str.replace(/dog/g, 'cat') // => 'My cat is not like other cats.'

The replace() method returns a new string with one, some, or all matches of a pattern replaced by a replacement. The pattern can be a string or a RegEx, and the replacement can be a string or a function called for each match. If pattern is a string, only the first occurrence will be replaced. The original string is left unchanged.

Syntax: replace(pattern, replacement) / Return value: String

replaceAll()

const str = 'My dog is not like other dogs.'

str.replaceAll('dog', 'cat') // => 'My cat is not like other cats.'

The replaceAll() method returns a new string with all matches of a pattern replaced by a replacement. The pattern can be a string or a RegExp, and the replacement can be a string or a function to be called for each match. The original string is left unchanged.

Syntax: replaceAll(pattern, replacement) / Return value: String

search()

const str = 'Hello World'

str.search(/W/g) // => 6

The search() method executes a search for a match between a regular expression and this String object.

Syntax: search(regex) / Return value: Index of the first match, or -1 if no match was found

slice()

const str = 'My dog is not like other dogs.'

str.slice(14) // => 'like other dogs.'
str.slice(3, 6) // => 'dog'
str.slice(-5) // => 'dogs.'
str.slice(-11, -6) // => 'other'

The slice() method extracts a section of a string and returns it as a new string, without modifying the original string.

Syntax: slice(indexStart) slice(indexStart, indexEnd) / Return value: String

split()

const str = 'My dog is not like other dogs.'

str.split(' ') // => ['My', 'dog', 'is', 'not', 'like', 'other', 'dogs.']
str.split(' ', 3) // => ['My', 'dog', 'is']
str.split('')[1] // => 'y'

The split() method takes a pattern and divides a String into an ordered list of substrings by searching for the pattern, puts these substrings into an array, and returns the array.

Syntax: split(separator) split(separator, limit) / Return value: Array

startsWith()

const str = 'Hello World'

str.startsWith('He') // => true
str.startsWith('He', 3) // => false

The startsWith() method determines whether a string begins with the characters of a specified string, returning true or false as appropriate.

Syntax: startsWith(searchString) startsWith(searchString, startPosition) / Return value: Boolean

substring()

const str = 'Hello World'

str.substring(6) // => 'World'
str.substring(6, 9) // => 'Wor'

The substring() method returns the part of the string from the start index up to and excluding the end index, or to the end of the string if no end index is supplied.

Syntax: substring(indexStart) substring(indexStart, indexEnd) / Return value: String

toLowerCase()

const str = 'Hello World'

str.toLowerCase() // => 'hello world'

The toLowerCase() method returns the calling string value converted to lower case.

Syntax: toLowerCase() / Return value: String

toUpperCase()

const str = 'Hello World'

str.toUpperCase() // => 'HELLO WORLD'

The toUpperCase() method returns the calling string value converted to uppercase.

Syntax: toUpperCase() / Return value: String

trim()

const str = '   Hello World   '

str.trim() // => 'Hello World'

The trim() method removes whitespace from both ends of a string and returns a new string, without modifying the original string.

Syntax: trim() / Return value: String

trimEnd()

const str = '   Hello World   '

str.trimEnd() // => '   Hello World'

The trimEnd() method removes whitespace from the end of a string and returns a new string, without modifying the original string. trimRight() is an alias of this method.

Syntax: trimEnd() / Return value: String

trimStart()

const str = '   Hello World   '

str.trimStart() // => 'Hello World   '

The trimStart() method removes whitespace from the beginning of a string and returns a new string, without modifying the original string. trimLeft() is an alias of this method.

Syntax: trimStart() / Return value: String

Array methods

method()

const arr = []

arr.method() // => []

Syntax: / Return value:

Object methods

method()

const obj = {}

obj.method() // => {}

Syntax: / Return value:
